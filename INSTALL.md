# Установка AnkiStreak на Samsung S25 Ultra

## Шаг 1: Сборка APK

### Вариант A: Android Studio (рекомендуется)

1. Скачай [Android Studio](https://developer.android.com/studio)
2. Открой проект: **File → Open** → выбери папку `anki-widget-analog`
3. Дождись синхронизации Gradle (2-5 минут при первом запуске)
4. Меню: **Build → Build Bundle(s) / APK(s) → Build APK(s)**
5. APK будет в: `app/build/outputs/apk/debug/app-debug.apk`

### Вариант B: Командная строка

Требуется JDK 17+ и Android SDK.

```bash
# Установи Android SDK (если нет Android Studio)
# Задай путь к SDK:
export ANDROID_HOME=$HOME/Android/Sdk
# Или создай local.properties в корне проекта:
echo "sdk.dir=$HOME/Android/Sdk" > local.properties

# Сгенерируй Gradle Wrapper (нужен gradle):
gradle wrapper --gradle-version 8.5

# Собери APK:
./gradlew assembleDebug

# APK здесь:
ls app/build/outputs/apk/debug/app-debug.apk
```

## Шаг 2: Перенос APK на телефон

Выбери один из способов:

| Способ | Как |
|--------|-----|
| **USB** | Подключи телефон кабелем, скопируй `app-debug.apk` в `Downloads` |
| **Telegram** | Отправь файл себе в "Избранное", открой на телефоне |
| **Google Drive** | Загрузи в Drive, скачай на телефоне |
| **Email** | Отправь себе на почту, скачай вложение |
| **ADB** | `adb install app-debug.apk` (если настроен ADB) |

## Шаг 3: Установка на Samsung S25 Ultra

1. Открой файл APK на телефоне (из загрузок / Telegram / etc.)
2. Система спросит разрешение: **"Установить из неизвестных источников?"**
3. Нажми **Настройки** → включи **"Разрешить из этого источника"**
4. Вернись назад, нажми **Установить**
5. После установки нажми **Открыть**

## Шаг 4: Настройка

### 4.1 Разреши AnkiDroid API

**ВАЖНО** — без этого виджет не сможет читать данные AnkiDroid!

1. Откройте **AnkiDroid**
2. Меню ☰ → **Настройки** → **Расширенные**
3. Найдите **"Разрешить доступ API"** → включите
4. Может появиться запрос на разрешение для AnkiStreak — подтвердите

### 4.2 Настрой AnkiStreak

В приложении AnkiStreak установи:
- **Цель карточек** — сколько карточек/день считать "занимался" (по умолчанию 10)
- **Время напоминания** — когда начинать напоминать (по умолчанию 12:00)
- **Критическое время** — когда отправлять срочное напоминание (по умолчанию 22:00)
- **Уведомления** — вкл/выкл

Нажми **Сохранить**.

### 4.3 Разреши уведомления

Samsung может заблокировать уведомления:
1. **Настройки** → **Приложения** → **AnkiStreak** → **Уведомления** → **Разрешить**
2. Также: **Настройки** → **Батарея** → **AnkiStreak** → **Не ограничивать**
   (чтобы Samsung не убивал фоновые процессы)

## Шаг 5: Добавить виджет на домашний экран

1. Зажми пустое место на домашнем экране (долгое нажатие)
2. Нажми **Виджеты**
3. Найди **AnkiStreak** в списке
4. Зажми виджет и перетащи на экран
5. Откроется окно настроек — нажми **Сохранить**
6. Готово! Виджет на экране.

## Шаг 6: Важные настройки Samsung (для надёжной работы)

Samsung One UI агрессивно убивает фоновые приложения. Чтобы виджет обновлялся и уведомления приходили:

### Отключи оптимизацию батареи для AnkiStreak:
1. **Настройки** → **Батарея и обслуживание** → **Батарея**
2. **Ограничения в фоне** → убедись что AnkiStreak **НЕ** в списке
3. Нажми на AnkiStreak → выбери **Не ограничивать**

### Отключи усыпление:
1. **Настройки** → **Батарея** → **Усыпление приложений**
2. Убедись что AnkiStreak **НЕ** в списке спящих приложений

### Закрепи в недавних:
1. Открой AnkiStreak
2. Перейди в недавние приложения
3. Нажми иконку AnkiStreak → **Закрепить** (или "Не закрывать")

## Как это работает

```
AnkiDroid Content Provider
        │
        ▼
   AnkiTracker ──── считывает due-карточки
        │            сравнивает с утренним снимком
        ▼            = reviewed_today
   StreakManager ─── если reviewed >= цель → стрик +1
        │
   ┌────┴────┐
   ▼         ▼
Widget    Notifications
(каждые    (12:00, 17:00,
30 мин)    20:00, 22:00)
```

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Виджет показывает 0 карточек хотя я занимался | Проверь что API доступ в AnkiDroid включён |
| Уведомления не приходят | Проверь настройки батареи Samsung (Шаг 6) |
| Виджет не обновляется | Удали и заново добавь виджет |
| "AnkiDroid не установлен" | Установи AnkiDroid из Play Store |
| Стрик не засчитался | Убедись что сделал >= порога карточек |
