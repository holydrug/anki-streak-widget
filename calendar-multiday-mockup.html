ПОЛНЫЕ ПРАВИЛА HYBRID V5
День с событиями → полный заголовок + список событий

1 пустой день → ghost-строка: "8 Вт" (opacity 10%, высота 22px)

2+ пустых подряд → диапазон: "10–11 Чт–Пт" (одна строка)

Пустая неделя целиком → week-strip (горизонтально 7 дней)
+ центральное сообщение "Свободная неделя"
+ карточка ближайшего события (если есть)

Пустая неделя, 1 событие далеко → week-strip с точкой на день события
+ "Сегодня: нет событий" + диапазон пустых + событие

Вообще пустой календарь → week-strip + "Нет событий · Календарь пуст"

Загруженная неделя → обычный agenda flow, ghost только для 1–2 пустых дней
АДАПТИВНАЯ ЛОГИКА
if (событий на неделе == 0) → week-strip + empty state + next event card
else if (событий на неделе <= 2 и все далеко) → week-strip сверху + agenda внизу
else → чистый agenda со scroll, ghost/range для пустых дней
